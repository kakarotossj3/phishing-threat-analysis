# Methodology

## Analysis Process

1. Email Header Analysis
   - Identify sender spoofing
   - Trace originating IP
   - Check SPF, DKIM, DMARC

2. Content Analysis
   - Social engineering techniques
   - Urgency and language patterns
   - Brand impersonation

3. Link & Domain Analysis
   - URL inspection
   - Domain age and reputation
   - Redirection behavior

4. Attachment Analysis (if applicable)
   - File type inspection
   - Sandbox execution

5. IOC Extraction
   - IPs, domains, URLs, hashes

6. MITRE ATT&CK Mapping
   - T1566 (Phishing)
   - Related techniques
