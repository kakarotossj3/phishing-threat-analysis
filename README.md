# Phishing Threat Analysis Lab

This repository documents ongoing analysis of phishing emails, focusing on detection techniques, attacker patterns, and mitigation strategies.

## Objectives

- Analyze real-world phishing attempts
- Identify Indicators of Compromise (IOCs)
- Document patterns and trends
- Improve detection and response strategies

## Structure

- /cases → Individual phishing cases
- methodology.md → Analysis methodology
- patterns.md → Observed trends
- tools.md → Tools used

## Disclaimer

All data has been sanitized to remove sensitive or personal information. This project is for educational and research purposes only.
